1.17 More Command Line Tools Page (17 mins )
  • pwd: print working directory
  • cp -r ./SourceFolder ./DestinationFolder: copy directory tree
  • find . : find in working directory
  • find . -name : name parameter
  • find test_creation -name "*file*" : wildcard find
  • grep "Hello" test_creation/* : grep will find within contents of file
  • grep -r "Hello" . : grep recursive find
  • grep -r "Hello" test_* : grep recursive wildcard find
  • mv ./SourceFolder ./DestinationFolder: move file
  • rm test_destruction/3.txt: delete file (only works on files)
  • rmdir test_destruction: delete directory (directory must be empty)
  • | : pass output from one command to another
  • function ff { find . -name $1; }: defined search parameter after 'ff'
